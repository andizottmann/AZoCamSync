Step 1: Prepare the Transcend WiFi SD card

In order to cooperate with this software on the SD card both an FTP and an TELNET service must be started. Therefore we need to install a autorun.sh script in the root directory of your card.
  • Put the WiFi SD card into the cardreader of this PC.
  • Choose the SD card by clicking on the button on the left side.
  • By pressing the OK Button in the File Dialog, the file will be installed.
